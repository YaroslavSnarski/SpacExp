Документация SpacExp
=====================

Добро пожаловать в документацию проекта **SpacExp** — системы для индексирования содержимого папок. SpacExp позволяет пользователям эффективно индексировать и организовывать файлы в папках с помощью графического интерфейса (GUI) или через веб-интерфейс. Это решение идеально подходит для управления файлами и быстрого поиска необходимой информации.

Особенности SpacExp:
====================

- Индексация файлов в папках (с поддержкой различных типов файлов).
- Поддержка интерфейсов: графического (GUI) и веб-страницы.
- Возможности для анализа, просмотра и управления метаданными файлов.
- Поддержка различных форматов документов и мультимедийных файлов.
- Удобный поиск по метаданным, таким как размер файла, дата создания и модификации, а также другие характеристики.

Система помогает организовать структуру файлов и упрощает доступ к необходимой информации.

Погрузитесь в детали разработки, просмотра и настройки SpacExp, следуя инструкциям ниже.

Как использовать SpacExp?
=========================

1. **Запуск приложения через GUI:**

   - Установите SpacExp и откройте приложение.
   - Выберите папку для индексации.
   - Запустите процесс сканирования.
   - Просматривайте результаты через графический интерфейс.

2. **Использование веб-интерфейса:**

   - Перейдите на страницу веб-интерфейса.
   - Загрузите файлы или выберите папку для индексации.
   - Просматривайте и управляйте файлами онлайн.
